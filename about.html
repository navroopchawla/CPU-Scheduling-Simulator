About the Project

Project overview

This CPU Scheduling Simulator is an Operating Systems virtual laboratory. It helps students and examiners visualize how processes are scheduled under FCFS, SJF, SRTF, Round Robin, and Priority policies — with Gantt charts, performance metrics, and exportable results.

Objectives

  • Implement standard CPU scheduling algorithms in a browser-based tool
  • Display Gantt-chart execution timelines for teaching and viva demos
  • Calculate waiting time, turnaround time, and CPU utilization
  • Provide theory documentation and report-ready templates

Modules

  • Home — Introduction and navigation
  • Documentation — Algorithm theory and advantages/disadvantages
  • Simulator — Input, Gantt chart, charts, CSV export
  • Report kit — See docs/PROJECT_REPORT_TEMPLATE.md and docs/VIVA_QUESTIONS.md

Tools & technologies

HTML5, CSS3, JavaScript, jQuery, Bootstrap, Chart.js, MathJax
